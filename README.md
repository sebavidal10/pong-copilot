# Pong Game 

This Pong game was generated by GitHub Copilot. 

## Overview 
The aim of the game is to control a paddle and hit a ball back and forth against an opponent's paddle. This implementation is a simple version of the classic Pong game, showcasing basic game mechanics.

## Features 
- Two player mode
- Simple and intuitive controls
- Score tracking
- Basic collision detection

## Controls 
- Player 1: Use W (up) and S (down) to control the left paddle.
- Player 2: Use the Up and Down arrow keys to control the right paddle.

## Running the Game 
To run the game, clone the repository and open the main game file in your preferred coding environment. Then, execute the program to start playing!
